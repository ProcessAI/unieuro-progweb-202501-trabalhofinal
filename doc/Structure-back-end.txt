A estrutura da nossa pasta back-end ficou neste formato:

/back -> nossa pasta raiz
|
|-- /src -> Aplicação principal do nosso projeto (geralmente app ou server)
|
|-- /routes -> Define as rotas da API (ex: /users, /auth)
|   |-- Route_clientes -> Mapeia os endpoints: GET, POST, PUT, DELETE, etc.
|
|-- /controller -> Recebe as requisições, chama os services, responde ao cliente
|
|-- /sevice -> Contém a lógica de negócio (acesso banco, validações, etc..)
|
|-- /database -> Onde fica o prisma + configuração do banco
|
|-- /node_modules -> Fic nossos pacotes instalados
|
|-- .gitignore -> Arquivos que o Git deve ignorar 
|
|-- package-lock.json -> Registra as versões exatas de TODAS as dependências e subdendências
|
|-- package.json -> Gerencia dependências e scripts
|
|-- tsconfig.json -> Condigurações do TypeScript
